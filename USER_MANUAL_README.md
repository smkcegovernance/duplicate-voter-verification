## Manual Contents

The user manual is specifically designed for urban teachers with limited technical knowledge and includes:

### 1. Introduction (परिचय)
- System overview
- Main objectives
- Target audience

### 2. System Access (प्रणालीमध्ये प्रवेश)
- How to open the application
- Browser requirements
- URL: https://smkc.gov.in:8443

### 3. Dashboard Understanding (डॅशबोर्ड समजून घेणे)
- Explanation of all statistics boxes:
  - Total records (एकूण नोंदी)
  - Verified records (प्रमाणित नोंदी)
  - Unverified records (अप्रमाणित नोंदी)
  - Duplicate records (डुप्लिकेट नोंदी)
  - Not duplicate records (डुप्लिकेट नाही)
  - Verification progress (प्रमाणित प्रगती)
- Navigation menu buttons

### 4. Voter Search and Verification (मतदार माहिती शोधणे आणि प्रमाणित करणे)
- Step-by-step instructions for searching voters
- **Important reminder**: Fill all fields in English only
- How to identify duplicate records
- How to mark duplicates
- How to mark non-duplicates
- Best practices and warnings

### 5. Report Generation (अहवाल तयार करणे)
- Using filters (all in English)
- Available filters:
  - Name fields (First, Middle, Last)
  - Ward/Div number
  - EPIC number
  - Serial number
  - Gender selection
  - Age range
  - Duplicate status
  - Verification status
- How to generate reports
- CSV export functionality

### 6. Troubleshooting (सामान्य समस्या आणि त्यांचे निराकरण)
Common problems and solutions:
- Page not loading
- Search not working
- Buttons not appearing
- Progress bar not showing
- CSV download issues

### 7. Important Rules (महत्वाचे नियम आणि सूचना)
DO's and DON'Ts:
- ✅ Check carefully
- ✅ Type in English
- ✅ Save regularly
- ❌ Don't rush
- ❌ Don't type in Marathi
- ❌ Don't select wrong records

### 8. Technical Information (तांत्रिक माहिती)
- System requirements
- Browser compatibility
- Internet requirements
- Security guidelines

### 9. Help and Contact (मदत आणि संपर्क)
- Technical support information
- FAQ section
- Common questions with answers
